move_mesh_vertex

compas_rhino.helpers.move_mesh_vertex(mesh, key, constraint=None, allow_off=None, redraw=True)[source]

Move on vertex of the mesh.

Parameters:

  • mesh (compas.datastructures.mesh.Network) – The mesh object.

  • key (str) – The vertex to move.

  • constraint (Rhino.Geometry) – Optional. A Rhino.Geometry object to constrain the movement to. Default is None.

  • allow_off (bool) – Optional. Allow the vertex to move off the constraint. Default is None.

Example

import compas
import compas_rhino as rhino

from compas.datastructures.mesh import Network

mesh = Mesh.from_obj(compas.get_data('lines.obj'))

key = compas_rhino.select_mesh_vertex(mesh)

if key:
    compas_rhino.move_mesh_vertex(mesh, key)
