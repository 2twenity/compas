.. currentmodule:: compas_rhino8.scene

This package provides scene object plugins for visualising COMPAS objects in Rhino 8.
When working in Rhino,
:class:`compas.scene.SceneObject` will automatically use the corresponding Rhino 8 scene object for each COMPAS object type.
